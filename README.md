# Stack

- Drizzle ORM - v1 beta https://orm.drizzle.team/docs/rqb-v2
- SQLite Database
- Container Docker Image
- Better Auth - https://better-auth.com/docs/introduction
- SvelteKit - https://svelte.dev/docs/kit/introduction
- oRPC and openAPI - https://orpc.dev/docs/getting-started, https://orpc.dev/docs/openapi/getting-started
- Tanstack/Svelte Query (client via oRPC) - https://tanstack.com/query/latest/docs/framework/svelte/ssr
- API Design using AEP - https://aep.dev/1/
